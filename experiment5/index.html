Description

What are you trying to acomplish with this experiment?
This is an interpretive art generator that artistically transforms an image into a variation made entirely of circles. The generator works best with high contrast images with clear depictions. It does not work well with small aspects in images as it is just a general geometric interpretation.


Technical

Describe in detail what you did technically and how it works?
In my_design.js, I generate, render, and mutate designs inspired by various images. The p4_inspirations() function returns an array of objects containing for the images. The p4_initialize() function initializes a design based on a selected inspiration, resizing the canvas. I also attempt to extract dominant colors from an image but is not currently utilized. The render function draws the design on the canvas using circles for each foreground element determined in the initialize function. The p4_mutate() function is there to allow the user to change the design by and adjusting properties of the images randomly with the help of mut()
